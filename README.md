# Banking System (OOP in C++)

A menu-driven banking system implemented in C++ demonstrating strong application of Object-Oriented Programming principles and real-world system design.

## Features

- Create Savings and Current Accounts
- Deposit and Withdraw operations
- Minimum balance constraint (Savings)
- Overdraft facility (Current)
- Multiple account handling
- Account deletion
- Data persistence using file handling
- Auto-load on startup and auto-save on exit

## OOP Concepts Used

- Encapsulation (private/protected data members)
- Inheritance (SavingsAccount, CurrentAccount)
- Polymorphism (virtual functions & overriding)
- Abstraction (base class design)

## Tech Stack

- Language: C++
- Concepts: OOP, File Handling, STL (vector)
